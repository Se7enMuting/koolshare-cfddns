# koolshare-cfddns
### 注意：
1. API Key：CF网站中My Profile里的Global API Key。
2. 不需要知道Zone ID和Domain ID，只需要填写域名即可。
3. 域名A记录：XXX.com或者YYY.XXX.com，不能是@.XXX.com。
4. TTL必须大于等于120（S）；或者填写1，表示自动。
